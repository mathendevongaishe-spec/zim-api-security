## Project Overview
This repository contains the cryptographic and financial modeling artefacts for my BSc Honours Dissertation. It evaluates Format-Preserving Encryption (FPE) as a risk mitigation strategy for third-party Fintech integrations.

## How to use the Simulation
The simulation performs the following:
- Generates 10,000 synthetic API transaction records.
- Conducts Jarque-Bera Normality and ADF Stationarity tests.
- Runs a 100,000-iteration Monte Carlo simulation to calculate VaR and Expected Shortfall.
- Compares FPE vs. Blockchain operational latency.
